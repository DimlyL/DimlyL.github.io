2023.5.20

发表于 2023-05-21 分类于 神秘

在学校又认真地学了一周的 whk，真的是几乎没去机房，因此连 Lovejoy 发新专都没注意到，这里面的话 CMWYL 是早些时候听过的，现在对 Consequences 和 Warsaw 蛮有好感。感觉他们与两年前的第一张专辑 Are You Alright 相比，在风格上已经有了很大的变化。虽然原本的也很好听，但我仍然也希望 Wilbur 这魔性的声音与 Lovejoy 的大家能带给我更多的惊喜。
从 Lovejoy 回来再谈谈 whk，我发现很痛苦，于是就偷偷赶时间之巅 5.0 的进度，把第九小节写完了，但是对于学考感到很头痛，于是决定接下来工作日要好好内卷，不能再颓废了。
但是在家就必须颓废！情人节与朝日过，520 与霧香过。今天又打了两个小时把《神明》的霧香线重通了。
从剧情意义上霧香线应该是最为巧妙的一条，霧香与黑江两人的意义对立与统一还需要我好好思考才能理清，但是我确实可以说这是加分点。也曾看到过有人对此评论说“能看出CUBE的野心，想要搞大黑深残，但最后只做出来一个这样的半成品”。我个人的意见是《神明》在本质上其实还是废萌、erogame 和 galgame，我觉得黑深残是不可能的，毕竟你要看受众群体和他们想玩的是什么。
另外还有在计划内的 CARD-GAME 代码重构和更多的 VOCALOID 计划。思考。

2023.5.13

发表于 2023-05-14 分类于 神秘

在 PKUSC 期间和回来后并没有写闲话，不过这个相关的话可以去洛谷上看游记。link。
回学校之后老老实实地学了一周的 whk，主要是在自习化学和历史，另外又滚回去研究时间之巅 5.0 了，目前又想要像写《列车少女》一样在 5.0 的世界观下写更多的短篇……这一回它是紫陰花的故事。话说这事紫陰花的第三个版本了吧，和云露一样明明不是很重要的角色但却被我眷顾着呢（）。
在学校里的时候发现 MORE MORE JUMP! 发新专了，是全世界漫游和变身领结的完整版！设了。仍然记得刚刚入坑国际服时就打了 World Wide Wander（全世界漫游）。
然后今天晚上在重推《献给神明般的你》，仍然记得四个月前我在博客里对于这款 gal 不公正的风评大发脾气。
说实话《神明 EX》让我很满意：作为拔作有着不言自明的强大，作为剧情作在 EX 版本中重做了问题最大的拉娜线，加入了点睛之笔的叶月线，剧情直接起飞。而且这玩意还有一个中意语音系统让我觉得很赞！
今天重推掉了月咏线。当时我是先推月咏再推霧香的，所以并没有完全搞懂为什么月咏的身份会在那场偶像活动的事故中暴露……现在知道了霧香眼睛里有摄像头了。
我还可以写很多关于《神明》的研究，但是比起那个我还想干脆发表一篇名为《论 Tinkle Position 及从〈妹禁手〉到〈童贞兄妹〉的发展历程和演变》的论文，这个东西我随手讲几千字。（？）
所以 Sister Position 快发新作，我相信你的童贞兄妹只是在憋大招。所以 Palette 快发新作，别惦记着你那炒冷饭了。

2023.4.30

发表于 2023-05-01 分类于 神秘

在家里快乐颓废了一天，并且终于是把评论系统搞出来了，Gitalk 牛逼，Waline 傻逼。
现在突然就不知道把这些独自一人的时间拿来干什么了。本来应该去赶稿时巅 5 的但是开摆了。
或许我们可以再推一个 gal，不过调色板的经典作品已经被我玩完了。接下来打什么，柚子社吗，虽然貌似是经典废萌但是我实在不想碰，柚子厨什么太令人讨厌了，虽然本身作品质量是高的。
或者说我们继续那个二刷计划，在糖调那里就莫名其妙中断了。怎么说呢，貌似废萌作确实是不大适合二刷的，因为剧情这种东西比较适合二刷但日常并不是，虽然也有例外，比如我觉得暴太郎或者齐木楠雄随便点开一集都很好。
说到剧情作那就得是《9-nine-》了，不过才刚玩完一个月立刻二刷是不是不大好。不过我好像没有其他手头的剧情作了。
所以还是要回去赶稿吗，哭。不如学 whk 或者好好准备 PKUSC，拿个约才是最现实的。

2023.4.29

发表于 2023-04-29 更新于 2023-04-30 分类于 神秘

今天推掉美樱线和杏线。怎么说呢……差强人意。美樱那个比较离谱，感觉剧情有点小浮夸，但也不算是烂吧。杏学姐的话，正常的双人格套路，这个我们在另外一个学姐香坂春风那里也体验过了，范围再大一点的话，远处的忍坂鸫也有着短暂的类双人格设定，不让 H 真的是一模一样？学姐线的剧情虽然很套路但是本身是没有问题的。但是我所期待艾蕾的 H scene 怎么只有口。过个五年调色板你炒冷饭的时候追加一下呗，顺便再写一条花子线。

另外今天《战神3》二周目通关，只能说不愧是经典游戏，放到现在来看仍然不输许多大作。再一次感受到了奎托斯身上所承载的痛苦、愤怒与从中涅槃重生的希望。
注意到由于时代限制，旧《战神》系列用的并非越肩视角而是全局的俯视。我必须夸赞圣莫尼卡在运镜方面是很厉害的，尤其要称赞奎托斯沿着奥林匹斯之链向上爬中途那个先放远再放近的镜头，震撼感足够强。还有在结尾被宙斯打入黑暗之中探索的这一部分的画面处理也相当好。
以及《战神3》的美术风格也让我相当喜欢，比如那个比较神秘的开场动画（Overtune），说实在的我不大懂这算是什么风格。

凌晨干碎缇娜线。注意了调色板这是废萌作，你再发刀子就把你干碎。
一件事两个刀，既干碎了当事人小春和爽又干碎了旁边的悠真和缇娜。你就说烦不烦吧。
不过好在结局缇娜又回来了，虽然有着很唯心的感觉但是此时此刻她能够回来的理由已经不重要了，重要的是她回来了！！！ohhhhhhhhhhhhhhhhh
以及我发现自己确实不是萝莉控。（？）
所以《恋花绽放樱飞时》被我光速干碎了。从整体上来看我觉得还算可以的，但是你废萌比不过《纯白交响曲》，剧情和刀子又比不过《9-nine-》，这个位置就很尴尬，调色板和田三部曲中地位最低了属于是（悲）。
不过从某种意义上这也是命中注定的，它想要传达的恋爱观就注定了它不能是一部美好的废萌作。《纯白》里面的恋爱是美好的，是各种颜色所组成的纯白交响曲；但是《恋花》是这样的东西。

无法恋爱的女孩子，对恋爱毫无兴趣的男孩子，这样的两人坠入爱河，世界会发生怎样的改变呢。

我再次引用游戏开头的这段话就是来说明性质。《恋花》里的恋爱更多地是苦涩的。能把这个主题展现的最完备的还是小奈美线和缇娜线。
我可以再引用缇娜线结尾的话，大概也是点明主旨了：

……

你妈的这个存档的 backlog 怎么回不去。看不到了，算了。下次再说。
这下又颓废了许久了。我 PKUSC 肯定爆炸！

2023.4.28

发表于 2023-04-28 分类于 神秘

我又不知道我这几天干了什么……哦，把《恋花绽放樱飞时》的小奈美线推掉了，巨难受无比。
主要是实妹的缘故。前面《纯白交响曲》由于樱乃是义妹所以有着很好的结局。但是在这里，虽然悠真和小奈美经过自己的努力让周围的人认可了他们之间跨越血缘的爱情，甚至有三年不能相见，但是这终究没有解决问题，在未来走上社会时他们还会因此受到更多的挫折。这并不是一个完美结局，让我烦躁。
但这也意味着无论未来遇上什么困难，他们都能共同渡过难关，这又令人感到温暖。
无法恋爱的女孩子，对恋爱毫无兴趣的男孩子，这样的两人坠入爱河，世界会发生怎样的改变呢。
游戏开头是这么写道的。不排除其他的线最后也是不完美结局……那我要直接整个烦死。
不过我希望其他人不会那么严重。小奈美的无法恋爱是因为是实妹，夕莉应该是风纪委员，美樱是对男性身体接触恐惧，杏不知道，缇娜更是不大懂，这俩好像和主线剧情有关。不过看起来还是小奈美最严重，也无法解决。

我花了一天推掉夕莉线，感觉还算是不错的，只是有点没搞懂关于休伯利安的事情。是纯粹因为休伯利安是男同才想要拆散两人，还是因为那是个类似于预言的东西，两人不分手花子就会死。如果是这样的话两人先分手再复合可不就是卡 bug 了。
夹击妹抖……真是好啊（流口水），反差萌做的够到位。夕莉好涩。还有调色板你为什么不做花子线，我觉得大家会喜欢的。
现在在推美樱线了。希望能是向安洁那样非常治愈的线路。
不过说到男性恐惧症就不得不提我们《9-nine-》的香坂春风了。虽然但是学姐有双人格，而且男性恐惧症大概也没有美樱那么严重，所以又不是一回事了。
顺带一提在学生会室里看到熊喵我直接立起来了。为什么（？）
